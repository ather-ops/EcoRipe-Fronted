## What Is EcoRipe

EcoRipe Frontend is a production-grade glassmorphism dashboard that connects to the EcoRipe ML backend API. It gives farmers real-time price predictions, 7-day forecasts, risk assessments, and harvest recommendations — across **12 apple varieties** — powered by a live machine learning model.

> **"From orchard to market — smarter decisions, better prices."**

- **12 Apple Varieties** — Gala, Fuji, Honeycrisp, Granny Smith, Red Delicious, Golden Delicious, Pink Lady, McIntosh, Braeburn, Empire, Cortland, Jonagold

- **User Authentication** — JWT-based secure login and registration
- **Email Collection** — marketing list builder with opt-in flow

### Visual and UI Features

- **Glassmorphism Cards** — transparent, frosted glass panels with blur effects
- **Animated Backgrounds** — floating orbs, dynamic gradient mesh, grid patterns
- **Dark / Light Mode** — full theme toggle with smooth CSS transitions
- **Interactive Charts** — live Chart.js visualizations fed from the ML API
- **Fully Responsive** — pixel-perfect on mobile, tablet, and desktop
- **Toast Notifications** — real-time success / error / warning alerts

### Authentication Flow

- JWT-based token authentication
- Secure password hashing on the backend
- Session management via localStorage
- Email opt-in captured at registration for marketing campaigns
- Protected routes — dashboard only accessible after login

## API Integration

```javascript
// Example: Fetch 7-day price prediction for a variety
async function getPricePrediction(variety) {
    const token = localStorage.getItem('ecoripe_token');
    const response = await fetch(`${API_BASE_URL}/api/predict`, {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
            'Authorization': `Bearer ${token}`
        },
        body: JSON.stringify({ variety: variety, days: 7 })
    });
    const data = await response.json();
    updatePriceChart(data.predictions);
    updateRiskMeter(data.risk_score);
    updateHarvestRecommendation(data.recommendation);
}
```
