# 🍎 EcoRipe Frontend - Apple Price Prediction Dashboard

![EcoRipe Logo](https://via.placeholder.com/150x150/0B3B2C/FFFFFF?text=ER)

## 📋 Overview
EcoRipe Frontend is a beautiful, glass-morphism dashboard that connects to the EcoRipe ML backend API. It provides farmers with real-time apple price predictions, risk assessments, and harvest recommendations across 12 different apple varieties.

## ✨ Features

### 🎯 Core Features
- **12 Apple Varieties**: Gala, Fuji, Honeycrisp, Granny Smith, Red Delicious, Golden Delicious, Pink Lady, McIntosh, Braeburn, Empire, Cortland, Jonagold
- **Real-time Price Predictions**: ML-powered 7-day price forecasts
- **Risk Assessment**: Visual risk meters with color-coded indicators
- **Harvest Recommendations**: AI-driven selling strategies
- **User Authentication**: Secure login/register system
- **Email Collection**: Build marketing lists for campaigns

### 📊 Visual Features
- **Glassmorphism UI**: Modern, transparent design with blur effects
- **Animated Backgrounds**: Floating orbs and dynamic grid patterns
- **Dark/Light Mode**: Toggle between themes
- **Interactive Charts**: Real-time data visualization with Chart.js
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop

### 🔐 Authentication
- JWT-based user authentication
- Secure password hashing
- Session management with local storage
- Email opt-in for marketing campaigns
